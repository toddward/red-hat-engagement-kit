# Architect Checklist

Pre-engagement preparation checklist for the lead architect.

## Technical Preparation

- [ ] Review customer's existing architecture documentation
- [ ] Understand engagement type and scope
- [ ] Review relevant Red Hat solution patterns
- [ ] Prepare discovery questionnaire based on engagement type
- [ ] Identify potential reference architectures

## Environment Access

- [ ] Customer environment access confirmed
- [ ] Lab or sandbox environment available (if needed)
- [ ] CLI tools installed (oc, kubectl, ansible, podman)
- [ ] SSH keys or credentials provisioned

## Tooling & Knowledge

- [ ] Engagement Kit repository cloned and configured
- [ ] `/setup` skill run to initialize engagement
- [ ] Knowledge base checklists reviewed
- [ ] Maturity model scoring criteria understood
- [ ] Previous engagement reports reviewed (if returning customer)

## Deliverable Planning

- [ ] Assessment output format agreed with customer
- [ ] Presentation template selected
- [ ] Timeline for deliverable drafts established
- [ ] Review cycle with customer sponsor planned
