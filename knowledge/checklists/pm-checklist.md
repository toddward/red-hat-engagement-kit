# Project Manager Checklist

Pre-engagement preparation checklist for the project manager.

## Administrative

- [ ] Statement of Work (SOW) signed and filed
- [ ] Purchase order confirmed
- [ ] Engagement dates finalized with customer
- [ ] Travel booked (if on-site)
- [ ] Expense approvals in place

## Stakeholder Coordination

- [ ] Customer sponsor identified and confirmed
- [ ] Key stakeholder list compiled
- [ ] Kickoff meeting scheduled
- [ ] Communication cadence agreed (daily standups, weekly reviews)
- [ ] Escalation path documented

## Access & Logistics

- [ ] Customer facility access arranged (badges, parking, building access)
- [ ] VPN/network access requests submitted
- [ ] Customer collaboration tools access (Slack, Teams, Jira, Confluence)
- [ ] Meeting room or workspace reserved

## Team Readiness

- [ ] Architect assigned and briefed
- [ ] Consultant(s) assigned and briefed
- [ ] Skills and deliverables scope reviewed with team
- [ ] Internal kickoff completed
- [ ] Knowledge base materials distributed to team
